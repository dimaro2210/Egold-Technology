CCTV Installation Course

Master the skills needed to design, install, and maintain CCTV systems for residential and commercial security applications.

Overview

CCTV (Closed-Circuit Television) is used for security, surveillance, and monitoring. Mastering CCTV installation gives you both a personal and business opportunity in today's security-conscious world.

Learning Objectives

Understand CCTV Systems

Learn about different types of CCTV systems and their applications

Identify Tools & Equipment

Recognize the tools and components needed for installation

Perform Installation

Execute a complete CCTV system installation from start to finish

Configure Remote Access

Set up systems for remote monitoring and access

Troubleshoot Issues

Diagnose and fix common CCTV system problems

Detailed Modules

Module 1: Introduction to CCTV

  • What is CCTV and how it works (camera → DVR/NVR → monitor)
  • Types of CCTV systems: Analog CCTV (DVR based), IP CCTV (NVR based, network cameras)

Module 2: Tools & Equipment

  • CCTV camera types: dome, bullet, PTZ
  • DVR/NVR recorder
  • Coaxial/ethernet cables, connectors (BNC, RJ45)
  • Power supply units
  • Monitor
  • Basic tools: drill, tester, crimping tool

Module 3: Installation Basics

  • Camera placement (avoid backlighting, ensure coverage)
  • Running cables (concealed vs open)
  • Connecting to DVR/NVR
  • Powering cameras

Module 4: Configuration

  • DVR/NVR setup (IP address, date/time)
  • Remote access (mobile app, port forwarding)
  • Testing and troubleshooting (no signal, unclear image)

Practical Tip

Start with a 4-camera DVR kit at home before going commercial.

Solar Installation Course

Master the skills needed to design, install, and maintain solar power systems for residential and commercial applications.

Overview

Solar energy is a clean, renewable source, widely used in homes, businesses, and rural electrification.

Learning Objectives

Understand Solar Systems

Learn how solar energy systems work and their components

Identify Components

Recognize all components of a solar power system

Install Systems

Install a complete off-grid solar system

Ensure Safety

Implement safety measures and maintenance procedures

Detailed Modules

Module 1: Introduction

  • How solar works (sunlight → solar panel → charge controller → battery → inverter → AC load)
  • Types of systems: Off-grid, On-grid, Hybrid

Module 2: Components

  • Solar panels (mono vs polycrystalline)
  • Charge controller (PWM vs MPPT)
  • Batteries (lead-acid, lithium)
  • Inverter
  • Mounting structures
  • Safety gear

Module 3: Installation Process

  • Site survey (roof angle, sunlight hours)
  • Mounting panels
  • Wiring: Panels → charge controller, Controller → battery, Battery → inverter → load
  • Grounding & surge protection

Module 4: Maintenance

  • Cleaning panels
  • Checking battery health
  • Monitoring system performance

Practical Tip

Begin with a small 1kVA system (1–2 panels, 1 battery) to learn hands-on.

Web Development Course

Learn to build responsive websites and web applications using HTML, CSS, JavaScript and modern frameworks.

Overview

Web development is the backbone of the digital world. Websites are needed by businesses, schools, NGOs, and individuals.

Learning Objectives

Understand Web Fundamentals

Learn how the web works and essential tools

Build Web Pages

Create structured web pages using HTML

Design Websites

Style websites with CSS and make them responsive

Add Interactivity

Implement functionality with JavaScript

Publish Websites

Deploy websites to live servers

Detailed Modules

Module 1: Getting Started

  • How the web works (browser, server, hosting)
  • Tools needed: text editor (VS Code), browser (Chrome/Firefox)

Module 5: Publishing

  • Free hosting: GitHub Pages, Netlify, Vercel

Phone Repair Course

Gain practical skills in diagnosing and repairing various smartphone hardware and software issues.

Overview

Smartphones are essential; phone repair is a profitable skill in every community.

Learning Objectives

Understand Phone Anatomy

Learn about phone components and common issues

Identify Tools

Recognize the tools needed for phone repair

Perform Hardware Repairs

Execute common hardware repairs

Perform Software Repairs

Fix common software issues

Detailed Modules

Module 1: Introduction

  • Phone anatomy: motherboard, battery, screen, charging port, camera
  • Common issues (won't charge, broken screen, water damage)

Module 2: Tools & Equipment

  • Screwdrivers (Torx, Phillips)
  • Multimeter
  • Soldering iron & hot air station
  • Plastic prying tools
  • Magnifying glass

Module 3: Basic Repairs

  • Replacing batteries
  • Screen replacement
  • Charging port replacement
  • Testing with multimeter

Module 4: Software Repairs

  • Flashing firmware
  • Removing FRP locks
  • Resetting phones

Practical Tip

Practice on old or dead phones before handling customer devices.

Cinematography Course

Learn the art of visual storytelling through camera operation, lighting, composition, and editing.

Overview

Cinematography is the art of storytelling through moving images. It combines creativity and technical skill.

Learning Objectives

Understand Camera Operation

Learn about different camera types and settings

Master Composition

Learn framing, composition and lighting techniques

Record Quality Audio

Learn to capture clear audio for videos

Edit Videos Professionally

Learn professional video editing techniques

Master Storytelling

Learn to tell compelling stories through video

Detailed Modules

Module 1: Introduction

  • Difference between videography & cinematography
  • Understanding camera types: DSLR, mirrorless, smartphone

Module 2: Basics of Shooting

  • Camera settings: ISO, shutter speed, aperture
  • Framing & composition (rule of thirds, leading lines)
  • Lighting basics (natural vs artificial)

Module 3: Audio

  • Importance of good sound
  • External mics vs inbuilt mic

Module 4: Editing

  • Software: CapCut, Premiere Pro, DaVinci Resolve
  • Basics: cutting, transitions, color grading

Module 5: Storytelling

  • Planning shots with storyboards
  • Shooting sequences (wide → medium → close-up)

Practical Tip

Start with a short 1-minute video project using your phone, then upgrade to pro gear.
